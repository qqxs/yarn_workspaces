## scripts 说明

```bash
# 启动 mobile app
# 首先已经安装 http-server
http-server -p ./package/mobile

# 启动 react spa app
yarn workspace react-spa-app start

# 启动 next.js ssr app
yarn workspace nextjs-app dev

# 启动 koa node server
yarn workspace koa-service-app start
```
